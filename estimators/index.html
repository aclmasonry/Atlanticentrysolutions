What to send

Four things and we can move.

  • Project name and bid close

    The closing date drives everything. If it is tight, say so — we would rather know up front than find out on the day.

  • The spec sections in scope

    Door and frame sections, hardware, glazing, sealing — whichever parts you want us on.

  • A link to the package

    Bid depository listing, Dropbox, SharePoint — a link beats attachments, and we can pull what we need from it.

  • Alternates you are weighing

    If something is already specified and you are checking whether one of our lines is a credible alternate, tell us and we will be straight about it.

Missing some of it is fine — send what you have and we'll work out the rest. Nothing here is a gate.
